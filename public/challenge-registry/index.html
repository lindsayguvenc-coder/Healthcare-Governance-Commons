Add your organization
Participation means sharing how you constructed your challenge set and ran your blind protocol — not patient data, not performance numbers. Other organizations running the same model can find you and request an introduction through the Commons. You control whether to respond.
What gets shared
  • Challenge set construction approach
  • Blind run protocol design
  • Performance comparison methodology
  • Lessons learned (500 chars)
  • Contact preference (direct / Commons-facilitated / none)
  • Patient data — never
  • Performance numbers — never
  • Patient-identifying information — never
Anonymization option
Organizations can contribute methodology without being identified by name. Registry entry reads as "Academic Medical Center, Urban, West Coast" or similar. Methodology is still usable. Contact requests route through Commons only.
How contact works
When you request an introduction, the Commons sends a blind introduction to the listed organization. They decide whether to respond. No contact information is published in the registry. You are never obligated to respond to requests.
Advocacy function
The Commons publishes an annual summary: how many organizations, which systems, what methodologies exist. This is the argument for formalizing proficiency testing programs in clinical AI. Registry participation makes that case. Read more in the Community Weaving series →
